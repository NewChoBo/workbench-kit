# Language Policy

영어를 공개 프로젝트 커뮤니케이션의 canonical 언어로 사용한다.

영어 [Language Policy](../../conventions/language-policy.md)가 canonical 문서입니다.

## 규칙

- 공개 문서, 커밋 메시지, release note, changelog, package metadata는 영어를 사용한다.
- 한국어 문서는 로컬 온보딩과 협업을 위한 companion 번역으로 제공할 수 있다.
- 영어 문서와 한국어 문서가 어긋나면 영어 문서를 먼저 수정하고 한국어 문서를 갱신한다.
- 과거 커밋은 언어 변경만을 위해 rewrite하지 않는다.
